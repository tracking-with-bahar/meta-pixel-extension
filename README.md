# Meta Pixel Inspector

This Chrome extension helps you see what’s really happening with Meta Pixels on any webpage.

If you work with tracking, ads, or analytics, this tool makes it easy to check whether a pixel is installed correctly and what data it’s sending.


## What it does

* Detects Meta Pixels on the current page
* Shows the Pixel ID
* Lists all events fired on the page
* Lets you expand each event to see:

  * Advanced matching data
  * Custom parameters
  * Event details like load time, referrer, page title, and event ID
* Lets you copy the Pixel ID with one click
* Includes a quick link to view the Meta Event Manager

## Why I built this

When you’re debugging tracking, you usually want quick answers:

* Is the pixel actually installed?
* Which events are firing?
* What parameters are being sent?
* Is advanced matching working correctly?

Instead of digging through the console or using heavy tools, this extension shows everything in a clean and structured way.

If you regularly work with Meta ads or conversion tracking, this tool is meant to make your workflow faster and clearer.
